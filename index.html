Xml4jQuery

Is a jQuery reactive plugin to supply XML, XPath and XSLT functionality.


   $(".toFill").xmlTransform( $.Xml(dataUrl).XPath("//book[@genre='romance']"), $.Xml(xslUrl) );
Look up the test sources for different use case.
Including into project xml4jquery.js is distributed as npm module and is available on CDN to be included as 

   <script type="text/javascript" src="https://cdn.xml4jquery.com/ajax/libs/xml4jquery/1.0.7/xml4jquery.js"></script>
Licensing It is a free to use and distribute binary library. The license agreement is embedded into xml4jquery.js file. If you are seeking for sources, it is easier to get open sourced AMD libraries also supported by XmlAspect.org
